開啟 MS-DOS 視窗

1. 放 Docerfile 檔案的目錄在 
   cd E:\Developers\SourceCodes\GitHub\Docker Project\Dockerfile\baseImages\kevinhung\custom-alpine-openssl


2. 執行 build docker images
   docker build -t kevinhung/custom-alpine-openssl:3.18.4 .
  
  
3. 查看 docker image 是否存在 tpebnkfiret 
   docker images | findstr custom-alpine-openssl
   
  
4. 將 images push 到 DockerHub
   docker push kevinhung/custom-alpine-openssl:3.18.4
 
 
5. 將 images 轉成 tar 存放
   docker save -o custom-alpine-openssl_3_18_4.tar kevinhung/custom-alpine-openssl:3.18.4
