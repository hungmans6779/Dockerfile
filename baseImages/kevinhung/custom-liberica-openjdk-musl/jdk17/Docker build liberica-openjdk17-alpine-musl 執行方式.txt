開啟 MS-DOS 視窗


1. 放 Docerfile 檔案的目錄在 
   cd D:\Developers\SourceCodes\GitHub\Docker Project\Dockerfile\baseImages\kevinhung\custom-liberica-openjdk-musl\jdk17

2. 執行 build docker images
   docker build -f DockerfileOpenJDK17 -t kevinhung/custom-liberica-openjdk-musl:17.0.9 .

3. 查看 docker image
   docker images | findstr custom-liberica-openjdk
   
4. 將 images 轉成 tar 存放
   docker save -o custom-liberica-openjdk-musl_17_0_9.tar kevinhung/custom-liberica-openjdk-musl:17.0.9

5. 將 images push 到 DockerHub
   docker push kevinhung/custom-liberica-openjdk-musl:17.0.9
