開啟 MS-DOS 視窗


1. 放 Docerfile 檔案的目錄在 
   cd E:\Developers\SourceCodes\GitHub\Docker Project\Dockerfile\baseImages\atmln\liberica-openjdk-alpine-musl


2. 執行 build docker images
   docker build -f DockerfileOpenJDK8 -t registry.groupt.fbt.com/tpebnkatmapt/custom-liberica-openjdk8-alpine-musl:3.18.4 .
   docker build -f DockerfileOpenJDK17 -t registry.groupt.fbt.com/tpebnkatmapt/custom-liberica-openjdk17-alpine-musl:3.18.4 .


3. 查看 docker image 是否存在 tpebnkfiret 
   docker images | findstr tpebnkfiret
   
   
4. 將 images 轉成 tar 存放
   docker save -o custom-liberica-openjdk8-alpine-musl_3_18_4.tar registry.groupt.fbt.com/tpebnkatmapt/custom-liberica-openjdk8-alpine-musl:3.18.4
   docker save -o custom-liberica-openjdk17-alpine-musl_3_18_4.tar registry.groupt.fbt.com/tpebnkatmapt/custom-liberica-openjdk17-alpine-musl:3.18.4
